# PWA Icons Setup

To complete the "Add to Home Screen" functionality, you need to add app icons.

## Quick Setup

1. **Open the icon generator**: Open `scripts/generate-icons.html` in your browser
2. **Generate icons**: Click the buttons to generate and download:
   - `icon-192.png` (192x192 pixels)
   - `icon-512.png` (512x512 pixels)
3. **Place icons**: Move the downloaded files to the `public` folder

## Alternative: Create Icons Manually

You can create the icons using any image editor:
- **icon-192.png**: 192x192 pixels
- **icon-512.png**: 512x512 pixels

Recommended design:
- Black or dark background with white "S" letter
- Or use your app logo/branding
